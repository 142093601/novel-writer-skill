# 类型套路库索引

> novel-analyzer 生成的类型内容分布在各 reference 文件中。本文件是索引，指向具体内容所在位置。

## 使用方法

1. 确定小说题材类型
2. 在本文件找到对应的 🏷️ 标记
3. 前往对应 reference 文件的 🏷️ 段落加载内容

## 素材加载策略

- **Planner 大纲阶段**：加载 plot-design.md + character-design.md + emotion-curve.md 中对应 🏷️ 段落
- **Writer 写作阶段**：加载 writing-techniques.md 中对应 🏷️ 段落
- **嵌入 Agent 任务时**：优先使用 slim/ 精简版，按需补充全量版

## 填充流程

1. 准备同类型小说文本（1-3篇）
2. 运行 novel-analyzer Mode B/C（Structure/Character/Style Analyst → Synthesizer）
3. 将 `## 🏷️ {类型}` 标签段落追加到对应 reference 文件
4. 生成 slim 精简版追加到 slim/ 目录
5. 更新本索引
