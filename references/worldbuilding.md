# 世界观构建指南

> 大纲阶段使用。根据题材选择对应模板。

## 核心原则

**世界观为剧情服务，不要过度设定。只保留能推动故事的部分。**

```
世界基础 → 核心冲突 → 关键地点 → 力量/规则体系 → 文化生活 → 绑定故事
```

## 现言/都市世界观速填

```
【时代背景】_____________（当代/近未来/90年代）
【社会阶层】_____________（豪门/中产/底层/多阶层混合）
【核心场景】_____________（公司/学校/家族/社区）
【权力结构】_____________（谁有钱/有权/有资源？）
【社会规则】_____________（潜规则/明规则/行业规则）
【阶层壁垒】_____________（如何跨越/为什么难以跨越）
【核心矛盾】_____________（阶层对立/贫富差距/权力斗争）
```

### 现言常见社会结构

**豪门文：**
```
统治层：家族掌门人（爷爷/父亲）
权力来源：血统 + 财力
核心资源：家族企业/遗产继承权
阶级：嫡系 > 旁系 > 外姓
核心矛盾：继承权之争/商业竞争/阶层偏见
```

**校园文：**
```
统治层：校董/校长/学生会
权力来源：家世 + 成绩 + 人气
核心资源：保送名额/社团资源/社交资本
阶级：学霸 > 普通生 > 差生（或按家世划分）
核心矛盾：成绩竞争/校园霸凌/阶层碰撞
```

**职场文：**
```
统治层：高管/老板
权力来源：职位 + 资源 + 人脉
核心资源：项目/晋升机会/客户关系
阶级：高管 > 中层 > 基层
核心矛盾：办公室政治/能力 vs 关系/性别偏见
```

**娱乐圈文：**
```
统治层：资本/顶级流量
权力来源：资本 + 粉丝量 + 作品
核心资源：好剧本/代言/热搜
阶级：顶流 > 一线 > 二线 > 十八线
核心矛盾：资源争夺/黑粉/潜规则/人设崩塌
```

## 力量体系设计要点

### 魔法/修炼体系
- **位阶明确**：分3-5个大段位，每段有代表性能力
- **代价体系**：越强的魔法/境界，代价越大（寿命/精神/反噬）
- **克制关系**：不同属性/流派之间有生克，避免一家独大
- **天花板设定**：前期就设定力量上限，防止后期崩坏

### 科技树
- **文明分级**：明确当前文明在什么等级
- **技术瓶颈**：关键技术卡在哪里，为什么突破不了
- **社会影响**：技术如何改变日常生活和权力结构

## 设定滴水呈现法

**原则：不要一次性倒出全部设定，像滴水一样每次释放一点。**

### 三种呈现方式

| 方式 | 适用 | 示例 |
|------|------|------|
| **对话带出** | 角色互动时自然透露 | "你不知道？金丹期的修士已经可以御剑了。" |
| **行动展示** | 通过角色行为展示规则 | 主角打碎杯子→碎片自动飞回→暗示这个世界有魔法 |
| **环境暗示** | 通过场景细节暗示世界观 | 街上有人骑鹤飞过→不用解释，读者自己懂 |

### 释放节奏

```
第1章：只展示读者需要知道的最小设定量（够理解剧情就行）
第2-3章：补充1-2个关键规则（通过事件触发）
中期：随剧情展开逐步揭示深层设定
后期：核心设定揭露（与高潮绑定）
```

### 避坑

❌ 开篇500字的世界观介绍 → 读者直接跳过
❌ 角色对话像百科全书 → 不自然
❌ 规则前后矛盾 → 建立设定文档，随时核对
❌ 过度设定与剧情无关的部分 → 只保留能推动故事的

---

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 力量崩坏 | 前期设天花板，明确代价 |
| 规则矛盾 | 建设定文档，定期核对 |
| 世界观空泛 | 只保留服务剧情的设定 |
| 过度设定 | 滴水式呈现，边写边展开 |
| 设定与剧情脱节 | 每个设定必须在剧情中有用 |

> 💡 不必填满所有模板，只设定与剧情相关的部分。世界观是舞台，故事才是主角。
