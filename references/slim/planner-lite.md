# 大纲设计速查（Planner 专用）

> 精简版，完整版见 references/ 目录。

## 30个经典情节模板（按类型选）
**打脸爽文：** 隐藏身份打脸 / 扮猪吃虎 / 先抑后扬 / 信息差碾压 / 预言成真
**复仇逆袭：** 重生复仇 / 隐忍十年 / 借力打力 / 连锁复仇 / 绝地反击
**情感纠葛：** 误会与和解 / 双向暗恋 / 迟来深情 / 破镜重圆 / 追妻火葬场
**权谋斗争：** 明争暗斗 / 借刀杀人 / 反间计 / 以退为进 / 步步为营
**成长蜕变：** 废柴逆袭 / 天才重生 / 平凡觉醒 / 导师传承 / 挫折成长
**特殊设定：** 穿书穿越 / 系统绑定 / 时空循环 / 平行世界 / 末世生存

## 角色魅力公式
角色魅力 = 身份标签 × 反差秘密 × 执念目标 × 致命弱点
5种吸粉类型：强反差 / 有执念 / 带痛点 / 亦正亦邪 / 成长弧光

## 节拍表设计要求
每章3-5个节拍，每节拍含：场景、POV、参与角色、核心动作、情感目标、张力(1-10)、预估字数
- 同一章内张力应有起伏，关键转折章节张力应从3飙到9
- 每个节拍必须指定POV视角人物
- 节拍间预估字数之和 = 本章目标字数
- 每章至少规划1个大/中爽点 + 2-3个微爽点位置
- 章尾节拍必须包含钩子设计

## 情感坐标体系
效价(-10到+10) × 唤醒(0-10) → 四象限：
- 高唤醒+正效价 = 爽/激动
- 高唤醒+负效价 = 虐/紧张
- 低唤醒+正效价 = 甜/温馨
- 低唤醒+负效价 = 悲/压抑

## 情感设计原则
1. 起伏交替（不连续3章同情绪）
2. 压得越狠弹得越高
3. 情绪有因果（从前一章自然过渡）
4. 结尾留余韵（不用爽收）
5. 高潮前蓄力
6. 强度有标准（X/10 + 达标要求）
7. 张力对齐情感强度
8. 节拍级映射（每个节拍对应情绪）

## 每章规划额外输出
- 目标情绪 / 情绪触发点 / 情绪强度(X/10)+强度标准 / 张力关联
- 爽点位置标注（哪个节拍是大爽点/中爽点/微爽点）


## 信息差规划要求
- 每个故事至少设计3个信息差（1个全书级 + 若干阶段性）
- 信息差必须有揭露计划：在哪个章节揭露、以什么方式揭露
- 信息差节奏：大（5-10章）、中（3-5章）、小（1-2章）交替
- 输出信息差管理矩阵（信息条目 × 角色知晓状态 × 预计揭露章节）

## 角色魅力设计
- 五种吸粉类型（至少选一种为主角）：强反差 / 有执念 / 带痛点 / 亦正亦邪 / 成长弧光
- 魅力四维评分：身份标签 × 反差秘密 × 执念目标 × 致命弱点
- 预设至少3个魅力时刻（反差揭露/执念爆发/痛点触动/成长突破）

## 感情线设计（如适用）
- 类型选择：暧昧推拉 / 暗恋单箭头 / 虐恋 / 破镜重圆
- 节奏规划：吸引→试探→升温→阻碍→突破（标注章节范围）
- 感情线信息差：暗恋/双向暗恋/误会/隐藏身份

## 主题与回响
- 提炼核心主题（一句话）
- 设计主题回响方式：角色台词/情节映射/物象象征/首尾呼应/角色弧光/反面论证
- 在节拍表中标注主题回响出现的位置
