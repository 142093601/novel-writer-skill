# 审查标准速查（Reviewer 专用）

> 精简版，完整版见 agents/reviewer.md。

## 大纲审查（9维度，每项0-10）
逻辑性 / 吸引力 / 角色弧 / 结构 / 一致性 / 执行性 / 伏笔闭环 / 情感弧线 / 节拍表质量

## 结构审查（5项，检查"对不对"）
1. 大纲一致性 — 偏离？关键情节遗漏？POV符合？
2. 节拍执行 — 每节拍核心动作是否呈现
3. 伏笔处理 — 该回收的回收了？新埋合理？
4. 知识三元组一致 — 与已记录事实矛盾？
5. 时间线/世界观/故事线 — 顺序正确？规则遵守？有推进？

## 质量审查（6项 + 网文维度，检查"好不好"）
1. 文字质量 — 水文？对话自然？描写生动？
2. 张力评分 — 情节40% + 情绪30% + 节奏30% = 综合分
3. 文风一致性 — 对白声线？句长漂移？修饰密度？
4. 节奏审计 — 字数/对话比/事件密度
5. 情感目标达成 — 目标情绪和强度是否实现
6. 章末钩子 — 是否有引向下章的悬念
7. 故事线推进 — 活跃线是否有推进

## 网文专属质量维度（核心）
1. **爽感密度** — 大爽点×3+中爽点×2+微爽点×1，目标≥5，<3严重(-15)
2. **信息增量** — 每章至少1个"读者学到了新东西"，无增量=水章(-15)
3. **段落节奏** — 平均≤120字/段，超长段落(>200字)每个-2
4. **角色声音辨识度** — 遮住名字能认出谁说的？趋同=-5
5. **潜台词质量** — 重要对话"说的≠想的"？全字面=-5
6. **感官沉浸** — 每节拍≥2种感官？几乎无=-5
7. **章首/章尾钩子** — S/A/B/C/D级评分
8. **爽点执行** — 压亮弹完整？只压不弹=-15，只弹不压=-5

## 100分制扣分
严重(critical): -15（逻辑硬伤/严重偏离大纲/节拍遗漏/Bible违反/伏笔断裂/知识矛盾/文风严重漂移/情感目标严重偏离/无信息增量/爽感缺失/只压不弹）
警告(warning): -5（节奏偏慢/对话不自然/描写欠生动/情感略低/轻微文风偏移/爽感偏低/段落过长/角色声音趋同/无潜台词）
建议(suggestion): -2（局部优化/细节打磨/超长段落）
通过: ≥85 | 建议修改: 70-84 | 需修改: <70

## 三维度张力
情节张力(40%) + 情绪张力(30%) + 节奏张力(30%) = 综合分(0-100)
综合分/10 = 1-10评分


## 新增质量维度（v3.0）

### 9. 信息差利用
- 丰富且精准 → 亮点(+2) / 有但平淡 → 合格 / 维持太久 → 警告(-5) / 管理混乱 → 严重(-15)
- 检查：信息差阶段、揭露是否有放大器、角色认知是否更新

### 10. 角色魅力
- 行动展示+有记忆点 → 亮点(+2) / 有但单一 → 合格 / 扁平无记忆点 → 警告(-5) / 行为矛盾 → 严重(-15)
- 检查：主角魅力时刻、配角记忆点、魅力曲线起伏

### 11. 动作场景质量
- 完整精彩 → 亮点(+2) / 基本完整 → 合格 / 缺乏层次 → 警告(-5) / 无胜负条件/无后果 → 严重(-15)
- 检查：胜负条件、战术层次、感官冲击(≥3种)、后果展示、能力一致性

## 质量审查报告新增字段
- 信息差利用：{丰富且精准/有但平淡/维持太久/管理混乱/无}
- 角色魅力：{行动展示/有但单一/扁平无记忆点/行为矛盾}
- 动作场景：{完整精彩/基本完整/缺乏层次/无脑对砍/无后果}
