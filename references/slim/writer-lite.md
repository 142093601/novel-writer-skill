# 写作技法速查（Writer 专用）

> 精简版，完整版见 references/writing-techniques.md + scene-writing.md。

## 开篇钩子（10种）
A. 绝境叠加+反常行动 / B. 极致选择题 / C. 地狱开局
D. 死亡回溯 / E. 身份反转 / F. 悬念前置（先抛结果回溯原因）
G. 时间炸弹（倒计时压力）/ H. 荒诞日常（反常细节制造诡异感）
I. 第一人称审判（主角直接对读者说话）/ J. 群像碰撞（多角色首幕引爆矛盾）

## 对话模式（10种）
①针锋相对 ②话里有话 ③沉默作答 ④打断抢白 ⑤自言自语
⑥对话框沉默（不回应也是一种回应）⑦旁人代述 ⑧回忆式对话 ⑨独白式对话 ⑩回声式

## 人物塑造核心
- 行动代替台词（颤抖的腿+破柴刀 > "她很勇敢"）
- 身体细节讲故事（疤痕连接童年创伤）
- 道具外化性格（锈钝柴刀="弱者也在保护自己"）
- 借角色吐槽套路（"人间清醒"人设）

## 爽点技法
**压-亮-弹：** 被压制→亮底牌（读者知道反派不知道）→反转碾压
**打脸三连击：** 反派挑衅→主角沉默示弱→一招秒杀→众人震惊
**身份揭露四步：** 铺垫被看不起→触发暴露→身份曝光→震惊链分层释放
**追妻火葬场：** 伤害→女主觉醒离开→男主卑微追悔→女主冷漠拒绝

## 节奏控制
开篇章：快 / 中间章：中 / 高潮章：快 / 结尾章：慢→留白
对话占比健康范围：25-45%

## 段落级铁律（核心）
- 每段 2-4 行（50-120字），超5行必须拆
- 段首钩子：动作先行/感官冲击/信息炸弹/对话开场/反差句/环境暗示
- 段尾惯性：悬念未解/情绪未平/动作未完/意外转折
- 冲击段独立成段（1句话），前后空行
- 每章至少1个信息增量（没有=水章）
- 信息逐步释放：Beat1给30%，Beat2给40%，Beat3给20%，Beat4留10%
- 每节拍至少2种感官描写，融入动作中
- 每节拍至少1个微爽点，每章爽感密度≥5

## 张力段落控制
| 张力 | 段落 | 句长 |
|------|------|------|
| 1-3 | 长段落 | 中长句 |
| 4-5 | 中等段落 | 中等句 |
| 6-7 | 短段落 | 短句为主 |
| 8-9 | 极短段落 | 碎片短句 |
| 10 | 单句段落 | 1-5字 |

## 角色声音指纹（5维度）
句式偏好 / 用词层次 / 语气标记 / 信息密度 / 情绪外显
→ 遮住名字能认出谁说的 = 合格

## 对话潜台词
角色说的 ≠ 想的。表面"路上小心"=实际"我不想你走"（配合动作）

## 章尾钩子强度
S级（重大反转）每5章 / A级（信息炸弹）每2-3章 / B级（悬念）每章 / C级（余韵）偶尔


## 信息差写法
- 每个节拍检查信息差状态（读者/角色/对手各知道什么）
- 信息差揭露要配放大器（旁观者震惊/环境变化/情绪爆发）
- 揭露时机：反派最嚣张时 / 配角绝望时 / 读者最期待时
- 揭露后角色认知必须更新（不能假装什么都没发生）

## 角色魅力展示
- 行动 > 台词 > 旁白（展示而非说明）
- 他人反应 > 自我评价
- 五种类型各有对应展示方式：强反差→反转 / 有执念→坚持 / 带痛点→脆弱时刻 / 亦正亦邪→意外选择 / 成长弧光→前后对比

## 动作场景铁律
- 五要素：胜负条件 + 情感赌注 + 战术层次 + 感官冲击(≥3种) + 后果可见
- 高张力(8-10)：短句密集，一段一个动作
- 低张力(1-3)：长句从容，战术分析
- 动作场景是信息差揭露的最佳载体

## 读者视角检查
- 读者现在在期待什么？是否满足或制造新期待？
- 读者对主角的感情是什么？（喜欢/心疼/着急）
- 读者接下来想看什么？是否给了继续读的理由？
- 读者会不会跳过这段？每段都有信息量吗？

## 配角出场铁律
- 每次出场必须有目的（推动情节/提供信息/制造冲突/展示主角魅力）
- 不能超过2章连续缺席后无交代
- 新配角出场需要记忆点（独特外形/标志性台词/特殊行为）

## power_changes 字段
- STATE_UPDATE 新增 power_changes 数组
- 记录：升级/获得/失去/封印/解封
- 战斗场景必须参照角色当前能力状态
