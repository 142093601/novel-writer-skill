# 情绪曲线设计参考

> 大纲阶段设计情感弧线，写作阶段对标执行。

## 核心原则

**情绪曲线 = 读者体验的心电图。**
好的小说让读者的情绪像过山车——有起有伏，有张有弛，但整体方向是向前的。

## 情绪坐标体系

用两个维度描述每个章节的读者情绪：

| 维度 | 范围 | 含义 |
|------|------|------|
| 效价 | -10 到 +10 | 正=愉悦/爽，负=憋屈/虐 |
| 唤醒 | 0 到 10 | 0=平静，10=极度激动 |

**情绪四象限：**
```
高唤醒 + 正效价 = 爽/激动/痛快（高潮打脸）
高唤醒 + 负效价 = 虐/愤怒/紧张（危机/背叛）
低唤醒 + 正效价 = 甜/温馨/治愈（日常/撒糖）
低唤醒 + 负效价 = 悲/压抑/绝望（低谷/反思）
```

## 标准情绪曲线模板

### 模板1：逆袭曲线（打脸/重生/复仇）
```
憋屈(高唤醒) → 积累(中唤醒) → 爽(高唤醒) → 平缓(低唤醒) → 更憋屈 → 更爽
  -8            -3            +8           +2           -9         +10
```
节奏：压→压→弹→喘口气→压得更深→弹得更高

### 模板2：甜宠曲线
```
甜(低唤醒) → 小虐(中唤醒) → 甜(高唤醒) → 大虐(高唤醒) → 超甜(高唤醒)
  +5          -4            +7           -8            +10
```
节奏：甜→虐→更甜→大虐→最甜。先给再夺再给更多。

### 模板3：悬疑曲线
```
好奇(中唤醒) → 紧张(高唤醒) → 突破(正) → 更好奇 → 更紧张 → 真相(极高唤醒)
  +3            -5           +4          +4        -7         ±10
```
节奏：悬念→压力→释放→新悬念→更大压力→终极真相

### 模板4：成长曲线
```
弱小(低唤醒) → 挫败(中唤醒) → 突破(中唤醒) → 更大挫败 → 蜕变(高唤醒)
  -2            -5           +5           -7          +9
```
节奏：低谷→打击→成长→更大打击→爆发

## 情绪落差设计

**爽感 = 落差 × 2。落差越大，爽感越强。**

| 模式 | 落差 | 爽感 |
|------|------|------|
| -3 → +5 | 8 | 中等爽 |
| -5 → +8 | 13 | 大爽 |
| -8 → +10 | 18 | 超爽 |
| -2 → +3 | 5 | 轻爽（日常用） |

**设计建议：**
- 大爽点前至少2章憋屈铺垫
- 每个爽点后安排1章情绪回落（给读者喘息）
- 全书至少2个"超爽"级别落差

## 情绪管理检查清单

### 每章检查
- [ ] 本章目标情绪是什么？
- [ ] 情绪强度达标了吗？（对照大纲要求）
- [ ] 与上章情绪是衔接还是反转？
- [ ] 读者读完本章应该有什么感受？

### 全书检查
- [ ] 情绪曲线有起有伏（不是全程平淡或全程高能）
- [ ] 至少2个情绪高峰（大爽点）
- [ ] 至少1个情绪低谷（为高潮蓄力）
- [ ] 结尾情绪高于开篇（整体向上）
- [ ] 章节间情绪落差不超过15（避免情绪断崖）

## 情绪与节拍的对应

| 节拍位置 | 建议情绪方向 | 说明 |
|----------|-------------|------|
| 章节开头 | 承接上章 | 平滑过渡，不突兀 |
| 节拍2-3 | 情绪推进 | 逐步升温或降温 |
| 章节高潮 | 目标情绪 | 达成本章情绪目标 |
| 章节结尾 | 留悬念 | 情绪未完全释放，引向下章 |

## 🏷️ 悬疑推理

> 来源：1篇悬疑推理小说《消失的第十一天》分析。大纲阶段使用。

### 标准情绪曲线

| 章节阶段 | 典型情绪 | 强度 | 趋势 |
|----------|----------|------|------|
| 开篇（第1章） | 恐惧+困惑 | 9/10 | 压 |
| 调查启动（第2章） | 愤怒+决心 | 6/10 | 积累 |
| 调查深入（第3章） | 紧张+不安 | 7/10 | 积累 |
| 突破（第4章） | 震惊+紧张 | 9/10 | 弹 |
| 推进（第5章） | 希望+兴奋 | 7/10 | 弹 |
| 高潮（第6章） | 紧张+义愤 | 9/10 | 弹 |
| 收尾（第7章） | 释然+满足 | 4/10 | 收 |

### 情绪节奏公式

**悬疑推理标准情绪节奏：**
- 开篇（第1章）：恐惧+困惑，强度9/10——地狱开局，读者与主角同时陷入信息黑洞
- 调查期（第2-3章）：愤怒→紧张，强度6-7/10——开始行动但充满不确定性
- 突破期（第4-5章）：震惊→希望，强度7-9/10——关键证据/记忆碎片推动真相
- 高潮（第6章）：义愤+紧张，强度9/10——真相对峙，正义与邪恶的终极碰撞
- 收尾（末章）：释然，强度4/10——情感收束，不用爽收，用释然/余韵

### 关键情绪转换点

1. **绝望→微弱希望**：从"铁证如山"到"鞋码不对"——第一个证据矛盾，读者信心首次倾斜
2. **希望→震惊**：催眠中看到搭档的脸——信任者变成嫌疑人，情感冲击峰值
3. **紧张→爽快**：真凶崩溃认罪——压抑了全篇的憋屈一次性释放
4. **爽快→释然**：正义实现后的平静——不是狂欢而是"终于结束了"的解脱感

## 🏷️ 玄幻修仙

### 标准情绪曲线（7阶段）

```
情绪值
  9 ─────── ★爆发────★爽快
  8 ─── 紧张╱          ╲
  7 ─────╱              ★震惊
  6 ──压抑                ╲蓄势
  5 憋屈
    └──Ch1──Ch2──Ch3──Ch4──Ch5──Ch6──Ch7──→
```

| 阶段 | 情绪值 | 对应章节 | 情绪关键词 | 写作手法 |
|------|--------|---------|-----------|---------|
| ① 憋屈 | 5 | 第1章 | 委屈、隐忍、麻木 | 环境烘托+沉默描写 |
| ② 压抑 | 6 | 第2章 | 愤怒被压、不甘 | 对话冲突+内心独白 |
| ③ 紧张 | 8 | 第3章 | 危机感、生死一线 | 节奏加快+悬念钩 |
| ④ 爆发 | 9 | 第4章 | 觉醒、释放、震撼 | 短句密集+力量描写 |
| ⑤ 爽快 | 9 | 第5章 | 碾压、打脸、畅快 | 连续爽点+震惊反应 |
| ⑥ 震惊 | 7 | 第6章 | 身份揭露、真相冲击 | 信息轰炸+多方视角 |
| ⑦ 蓄势 | 6 | 第7章 | 意犹未尽、新悬念 | 安静收尾+钩子抛出 |

---

### 情绪节奏公式

**核心公式**：`压抑时长 × 2 = 爆发强度`

| 公式 | 含义 | 应用 |
|------|------|------|
| 压抑时长 × 2 = 爆发强度 | 压抑越久，爆发越爽 | 前3章压抑→第4章高爆发 |
| 爽点间距递减 | 越往后爽点越密集 | Ch5每半章1个爽点 |
| 高潮后必须回落 | 不可持续高强度 | Ch6震惊(7)→Ch7蓄势(6) |

**情绪配比建议**（7章短篇）：

| 情绪区间 | 章节数 | 占比 |
|---------|--------|------|
| 低情绪（5-6） | 3章 | 43% |
| 中情绪（7-8） | 2章 | 28% |
| 高情绪（9） | 2章 | 29% |

> **黄金比例**：低:中:高 ≈ 4:3:3
> 低情绪铺垫不足，高情绪爽感打折。

---

### 关键情绪转换点

#### 转换点一：憋屈 → 紧张（第2章末→第3章）

- **触发事件**：被迫参赛/遭遇生死危机
- **技法**：节奏突变，从慢描写切换为快叙事
- **标志**：章末悬念钩，"她不得不上台。"

#### 转换点二：紧张 → 爆发（第3章末→第4章）

- **触发事件**：觉醒触发（前世记忆/灵力暴走/生命威胁）
- **技法**：短句爆发，力量描写，感官放大
- **标志**：一击制敌，全场死寂
- **示例**："那一瞬间——她想起了所有的事。"

#### 转换点三：爽快 → 震惊（第5章→第6章）

- **触发事件**：身份真相揭露
- **技法**：从"打脸快感"切换为"命运沉重感"
- **标志**：情感反转（爽→沉）
- **示例**："赢了又怎样？千年前那个人……已经不在了。"

#### 转换点四：震惊 → 蓄势（第6章→第7章）

- **触发事件**：真相消化后，新悬念出现
- **技法**：节奏放缓，安静场景，暗示未来冲突
- **标志**：章末钩子指向新篇章
- **示例**："她望向远方——那里，有什么东西正在苏醒。"

---

### 情绪管理检查清单

写完每个章节后，对照检查：

- [ ] 本章情绪值是否符合曲线定位？
- [ ] 压抑章节是否"压够了"？（读者是否足够憋屈）
- [ ] 爆发章节是否有足够的前置铺垫？
- [ ] 爽点之后是否有情绪回落？（不可持续高强度）
- [ ] 章末是否有钩子？钩子是否指向下一情绪阶段？
- [ ] 是否有"情绪断崖"？（突然跳跃无过渡）
## 🏷️ 穿书反套路

> 来源：1篇穿书小说《穿成恶毒女配后，我躺平了》分析。大纲阶段使用。

### 标准情绪曲线

| 章节阶段 | 典型情绪 | 强度 | 趋势 |
|----------|----------|------|------|
| 穿越觉醒（第1章） | 困惑+喜感 | 6/10 | 压 |
| 躺平策略（第2章） | 轻松+好笑 | 5/10 | 弹 |
| 角色异动（第3章） | 暧昧+好奇 | 6/10 | 积累 |
| 先知避祸（第4章） | 紧张+兴奋 | 7/10 | 压 |
| 闺蜜建立（第5章） | 温馨+感动 | 5/10 | 收 |
| 甜宠收束（第6章） | 甜蜜+纠结 | 6/10 | 弹 |
| 大团圆（第7章） | 满足+躺赢 | 3/10 | 收 |

### 情绪节奏公式

**穿书反套路标准情绪节奏：**
- 开篇：困惑+喜感6/10——穿越觉醒的反差喜剧
- 中段：轻松→紧张→温馨5-7/10——躺平策略+先知避祸交替
- 收尾：甜蜜→满足6-3/10——甜宠收束，不用大爽收，用温馨满足

### 关键情绪转换点

1. **困惑→轻松**：决定躺平不作妖——从"怎么办"到"不管了"
2. **轻松→紧张**：商业危机来临——先知优势面临考验
3. **紧张→温馨**：危机化解+闺蜜建立——外部压力释放+关系深化
4. **温馨→甜蜜**：男主正式追求——爱情线收束

## 🏷️ 虐文/追妻火葬场

> 来源：1篇虐文小说《他的白月光回来后，我成了前妻》分析。大纲阶段使用。

### 标准情绪曲线

| 章节阶段 | 典型情绪 | 强度 | 趋势 |
|----------|----------|------|------|
| 表象幸福（第1章） | 憋屈 | 7/10 | 压 |
| 尴尬相处（第2章） | 压抑 | 6/10 | 压 |
| 钝刀割肉（第3章） | 心冷 | 7/10 | 压 |
| 心碎确认（第4章） | 心碎 | 9/10 | 弹 |
| 干净离开（第5章） | 决绝 | 8/10 | 弹 |
| 追妻火葬场（第6章） | 释然 | 7/10 | 收 |
| 开放结局（第7章） | 平静 | 5/10 | 收 |

### 情绪节奏公式

**追妻火葬场标准情绪节奏：**
- 虐段（第1-4章）：憋屈→压抑→心冷→心碎，强度7→9，高台跳水式爬升
- 转折（第5章）：决绝8/10——干净离开的标志性时刻
- 爽段（第6章）：释然7/10——追妻火葬场的报复性快感
- 收尾（第7章）：平静5/10——释然比爽更有余韵

### 关键情绪转换点

1. **甜蜜→憋屈**：三周年晚宴白月光消息——甜蜜节点变崩塌起点
2. **隐忍→心冷**：多次偏袒累积——钝刀割肉式心死
3. **心冷→心碎**：发现书房照片——物证击碎最后幻想（全篇情绪峰值）
4. **心碎→决绝**：提出离婚——从被动承受者变主动决定者
5. **决绝→释然**：拒绝追悔——权力完全反转
