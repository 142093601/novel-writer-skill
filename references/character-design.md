# 角色设计参考

> 大纲阶段必须使用。确定角色时按模板填写。

## 角色魅力公式

```
角色魅力 = 身份标签 × 反差秘密 × 执念目标 × 致命弱点
```

**5种吸粉角色类型：** 强反差 / 有执念 / 带痛点 / 亦正亦邪 / 成长弧光

**立住角色3要素：** 核心欲望（全书只盯一个）/ 致命弱点（冲突来源）/ 行为标签（口头禅/小动作）

> 让人上瘾的角色 = 强烈欲望 + 致命弱点 + 清晰成长 + 独特反差

## 角色档案模板

### 基础版（配角）
```
姓名 / 年龄 / 外貌（1个记忆点）/ 身份（明面+隐藏）
性格反差：外表__ vs 内心__
核心动机：最想要__ / 最害怕__
致命弱点 / 口头禅 / 与主角关系
```

### 完整版（主角）
```
【身份】核心标签 / 姓名（本名+别名）/ 性别年龄 / 明面+隐藏身份
【外在】基本体貌 / 标志性特征（唯一视觉锚点）/ 衣着 / 仪态 / 声音 / 口头禅
【内在】核心性格+次要特质（矛盾面）/ 价值观 / 优点 / 致命弱点 / 内在矛盾
【背景】成长环境 / 3个关键转折事件 / 隐性创伤或执念
【驱动】短期目标 / 长期追求 / 最害怕 / 最想要
【成长】初始状态 → 转折 → 挣扎突破 → 终点蜕变
【关系】最亲/最恨/最亏欠/官配对手/导师/追随者
```

### 反派专用
```
核心标签 / 姓名 / 表面身份 / 真实身份
反派动机：为什么作恶？为什么针对主角？有什么苦衷？
能力资源 / 致命弱点 / 结局
```

## 关系模式库

### 10种核心关系类型

| 类型 | 特征 | 弧光可能 |
|------|------|----------|
| 主导型 | A掌控B | 掌控→反抗→平等/反转 |
| 从属型 | B依附A | 依附→独立→对等 |
| 平行型 | A/B各自独立 | 独立→碰撞→融合 |
| 互补型 | A有B缺的，B有A缺的 | 互补→冲突→更深互补 |
| 对抗型 | A和B立场对立 | 对抗→理解→和解/亦敌亦友 |
| 羁绊型 | 命运绑定，无法切割 | 纠缠→挣脱→选择 |
| 镜像型 | A和B是同一枚硬币的两面 | 镜像→认同/排斥→蜕变 |
| 守护型 | A单向保护B | 守护→被守护→双向 |
| 交易型 | 基于利益的临时合作 | 交易→信任→真情/背叛 |
| 轮回型 | 前世/上辈子的因果延续 | 重复→打破→超越 |

## 角色语言区分指南

**核心原则：两个角色说同一句话，读起来应该不一样。**

### 区分维度

| 维度 | 变化范围 | 举例 |
|------|----------|------|
| 句式长短 | 短句型 vs 长句型 | "行。" vs "这个事情呢，我觉得还是要考虑一下的。" |
| 用词层级 | 口语/文雅/专业/方言 | "搞定" vs "处置妥当" vs "close掉" |
| 语气倾向 | 陈述/反问/祈使/感叹 | "你去。" vs "你不会不去吧？" vs "去！" |
| 标志词 | 口头禅/语气词 | "说真的" / "嗯哼" / "不是我说" |
| 信息密度 | 啰嗦 vs 精炼 | 三句话说一件事 vs 半句话点破 |
| 情绪表达 | 外放 vs 内敛 | 拍桌子 vs 面无表情 |

### 角色声线速查卡

设计角色时，为每个主要角色填写：

```
【角色名】声线卡
- 句式：短（5字内）/ 中（10字左右）/ 长（15字+）
- 用词：口语 / 书面 / 专业 / 特定领域
- 语气：强势 / 弱势 / 调侃 / 冷漠 / 温柔
- 口头禅："{固定表达}"
- 禁区：绝不会说的话（如：绝不道歉 / 绝不示弱）
- 典型台词："{最能体现声线的1-2句}"
```

### 示例：同一场景不同声线

场景：主角被问"你为什么不去？"

| 角色类型 | 台词 |
|----------|------|
| 强势型 | "不想去。"（短句+陈述+终结话题） |
| 调侃型 | "去了怕你们受不了我。"（反讽+自信） |
| 内敛型 | "……有点远。"（回避+借口） |
| 毒舌型 | "因为看到你们就够了。"（攻击性+幽默） |
| 温柔型 | "下次吧，这次真的有事。"（委婉+歉意） |

## 角色弧光模板

**成长型弧光：**
- 起点：{弱/天真/逃避/封闭}
- 转折：{关键事件打破舒适区}
- 挣扎：{在新旧自我间摇摆}
- 终点：{强大/成熟/面对/开放}

**觉醒型弧光：**
- 起点：{被蒙蔽/顺从/工具人}
- 触发：{真相揭露/背叛/关键对话}
- 觉醒：{认知重建}
- 行动：{做出不同选择}

**堕落型弧光：**
- 起点：{善良/理想主义}
- 侵蚀：{一步步被现实/欲望改变}
- 临界：{不可逆的选择}
- 终点：{面目全非}

**扁平型（不成长）：**
- 适用：功能性角色、反派、背景人物
- 特征：全程保持一个特质，作为主角的对照

## 角色出场管理

**重要角色不能"失踪"：**
- 主角：每章必须出场
- 核心配角（3-5人）：最多缺席2章
- 次要配角：每5章至少提一次

**缺席时的维持存在感：**
- 其他角色提及
- 遗留物品/信件/消息
- 回忆/闪回
- 环境线索（他的座位空着）

## 避雷指南

❌ 完美无缺的圣人 → 给致命软肋
❌ 全程工具人 → 给独立目标和情感线
❌ 堆优点 → 只给1个标志性特征
❌ 性格突变 → 成长要有依据和过程
❌ 脸谱化反派 → 要有智商和动机
❌ 所有角色说话方式一样 → 用声线卡区分

## 让角色"破例"制造苏点
从不低头→为你低头 / 从不救人→只救你 / 从不解释→对你解释
**破例 = 苏点 + 虐点 + 名场面**

## 🏷️ 重生复仇 角色原型

> novel-analyzer 分析生成。来源：2篇重生复仇小说。

### 必备角色（3个）

**重生复仇女主** — 承载读者代入，执行复仇计划（10/10篇）
- 特征：隐忍、果决、擅长伪装、信息优势（重生记忆）
- 语言：短句为主，表面温柔内心狠辣，内心独白用反问
- 口头禅："这一世"、"前世的我"、"可惜"、"等着瞧"
- 典型台词：
  - "这一世，我绝不会让你们得逞。"
  - "林晚，你的演技，能骗我多久？"
  - "我笑了笑，心里却在冷笑。"

**渣男未婚夫** — 制造冲突，提供打脸对象（9/10篇）
- 特征：虚伪、贪婪、表面深情、精于算计
- 语言：温柔长句（假面期）→ 短句推卸（暴露后）
- 典型台词：
  - "念念，我会一辈子对你好的。"
  - "不管你有没有钱，我都爱你。"

**白莲花女配** — 表面甜实际毒，制造情感冲突（8/10篇）
- 特征：表面乖巧、内心恶毒、演技好、野心大
- 语言：甜腻语气（伪装期）→ 尖酸刻薄（暴露后）
- 典型台词：
  - "姐姐今天真漂亮，姐夫你可要好好珍惜姐姐啊。"
  - "子墨从来就没爱过你，你知道吗？"

### 可选角色
- **忠犬男主** — 正牌男主，提供情感支持（7/10篇）
- **恶毒继母** — 幕后推手（6/10篇）
- **正义盟友** — 律师/老员工（5/10篇）

### 关系模式
| 模式 | 角色A | 角色B | 关系弧线 | 触发 | 频率 |
|------|-------|-------|----------|------|------|
| 渣男+女主+白莲花三角 | 重生女主 | 渣男+白莲花 | 信任→背叛→揭露→毁灭 | 重生 | 8/10篇 |
| 女主+忠犬守护 | 重生女主 | 忠犬男主 | 陌生→帮助→并肩→相爱 | 关键帮助 | 7/10篇 |
| 女主+恶毒继母 | 重生女主 | 恶毒继母 | 伪装→识破→交锋→揭露 | 发现真面目 | 6/10篇 |

### 角色弧光：复仇觉醒型
- 起点：天真善良，被蒙蔽
- 转折：被背叛致死，重生
- 过程：觉醒→冷酷→找到平衡
- 终点：强大但不失去人性

## 🏷️ 甜宠 角色原型

> novel-analyzer 分析生成。来源：1篇甜宠小说。

### 必备角色（2个）

**甜宠女主（小透明逆袭型）** — 从自卑到被爱（9/10篇）
- 特征：接地气、自嘲型幽默、善良、努力
- 语言：短句+碎碎念，内心戏多，吐槽感强
- 口头禅："干！"、"冷静"、"好家伙"
- 典型台词：
  - "干！"
  - "冷静，苏晚晚，你要冷静。这只是演戏。只是演戏！"
  - "好家伙，她在人设里居然是个恋爱脑舔狗？"

**暗恋型男主（高冷+深情）** — 提供极致甜宠（9/10篇）
- 特征：外冷内热、行动派、暗恋多年、不善表达
- 语言：极短句（2-5字），偶尔温柔长句
- 口头禅："进来"、"不用演"、"不可能找不到"
- 典型台词：
  - "进来。"
  - "你就是你，不用演。"
  - "不可能找不到。因为我会一直找。就算找一辈子，我也会找到你。"

### 可选角色
- **毒舌经纪人** — 喜剧元素+推动剧情（6/10篇）
- **反派情敌** — 制造小冲突，甜宠文中通常弱化（5/10篇）

### 关系模式
| 模式 | 角色A | 角色B | 关系弧线 | 触发 | 频率 |
|------|-------|-------|----------|------|------|
| 合约恋爱变真爱 | 甜宠女主 | 暗恋型男主 | 假恋爱→同居→暗糖→掉马甲→真爱 | 合约开始 | 8/10篇 |
| 小透明逆袭 | 甜宠女主 | 外界 | 被看不起→被宠爱→众人震惊 | 公开恋情 | 7/10篇 |

### 角色弧光：被爱觉醒型
- 起点：自卑/不相信自己值得被爱
- 转折：男主持续温柔行动
- 过程：心动→纠结→确认→敞开心扉
- 终点：自信且被深爱

## 🏷️ 悬疑推理

> 来源：1篇悬疑推理小说《消失的第十一天》分析。大纲阶段使用。

### 标准角色配置

**悬疑推理必备角色（3个）：**
1. **冤屈的猎犬** — 被冤枉的专业人士，凭专业技能从绝境中翻盘
   - 典型特征：专业能力极强、身处绝境、不放弃、用能力自证
   - 出现率：核心
2. **披着羊皮的狼** — 最亲近的人是凶手，完美伪装的反派
   - 典型特征：长期伪装、利用信任、有不可告人的过去
   - 出现率：核心
3. **专业辅助者** — 提供关键技术支持的功能性角色（律师/法医/技术员）
   - 典型特征：专业、中立、提供突破口
   - 出现率：核心

**悬疑推理可选角色（2个）：**
1. **狱中先知** — 囚禁环境中出现的智慧型盟友，提供独特视角（5/10篇）
2. **沉默的牺牲者** — 因知道太多被灭口，死后仍驱动剧情（7/10篇）

### 角色语言特征模板

**冤屈的猎犬说话方式：**
- 句式：短句+反问句推动思考
- 用词：刑侦术语+日常口语混合
- 口头禅："不对...这里有问题"、"等等"、"你是说..."
- 典型台词：
  - "我是刑警，我怎么会杀人？而且我不记得我杀过人！"
  - "这个照片里的人穿着运动鞋，我从来不穿运动鞋。"

**披着羊皮的狼说话方式：**
- 句式：温柔长句为主，关键时刻沉默
- 用词：关心+专业的表面话
- 典型台词：
  - "我永远站在你这边。"
  - "不管发生什么，我都会帮你。"

**狱中先知说话方式：**
- 句式：简洁直白，偶尔带黑色幽默
- 用词：犯罪世界的行话+生活化表达
- 典型台词：
  - "刑警坐牢，有意思。"
  - "你的眼神，是一个无辜者的眼神。"

### 关系模式库

| 模式 | 角色A | 角色B | 关系弧线 | 典型触发 | 频率 |
|------|-------|-------|----------|----------|------|
| 搭档变仇敌 | 冤屈的猎犬 | 披着羊皮的狼 | 信任→猜疑→对立 | 发现证据矛盾指向搭档 | 核心 |
| 绝境中的意外盟友 | 冤屈的猎犬 | 狱中先知 | 陌生→试探→信任合作 | 狱友身份+共同分析 | 特色 |
| 好友灭口 | 披着羊皮的狼 | 沉默的牺牲者 | 好友→发现秘密→灭口 | 发现旧案真相 | 常见 |

### 角色弧光模板

**牢笼中的猎犬：**
- 起点：失忆+被控谋杀，双重打击，最脆弱
- 转折：律师带来案件资料，开始用刑侦本能分析
- 过程：在牢房中重建证据链，每章发现一个疑点
- 终点：以逻辑和证据推翻指控，揭露真凶是搭档

## 🏷️ 玄幻修仙

### 标准角色配置

#### 必备角色（3人核心组）

**① 废柴觉醒女主**
- **原型**：fallen_hero_rebirth
- **核心特质**：隐忍 → 善良 → 觉醒后碾压
- **外在表现**：前期沉默寡言、忍辱负重；觉醒后气场全开、言语犀利
- **内在驱动**：守护之人/复仇执念/找回身份
- **经典出场**："她蹲在药田边洗灵草，洗了三年，没人记得她叫什么。"

**② 踩脚石反派**
- **原型**：petty_bully_stepping_stone
- **核心特质**：仗势欺人、外强中干、一击即溃
- **功能定位**：为主角提供"打脸"机会，衬托主角实力
- **经典台词**："就你这种废物，也配站在我面前？"

**③ 深藏不露男主**
- **原型**：hidden_protege_male_lead
- **核心特质**：表面普通/低调，实则有特殊身份（宗门少主/上古血脉/隐世强者）
- **功能定位**：暗中保护女主，关键时刻出手，身份逐步揭露
- **经典出场**：人群中不起眼，但女主每次危险时"恰好"出现

#### 可选角色（1人增益组）

**④ 背叛者**
- **原型**：traitor_betrayer
- **核心特质**：前世最信任的人，转世后以新身份接近，实则别有用心
- **功能定位**：制造情感冲击，提供"前世复仇弧"驱动力
- **经典揭示**："你前世最信任的那个人——就是我杀的。"
